Arrangement.Constrained

This arrangement shows featured cards that follow a irregular pattern in three different card sizes. The pattern is described using Emeus VFL (Visual Format Language).

NOTE that this arrangement design is still experimental therefore it visuals and behavior are subject to change.

This arrangement can be extended by programmers to implement other constrained arrangements by simply providing a VFL description and the number of cards.

To extend this class override the Arrangement.get_max_cards and Constrained.get_description.
